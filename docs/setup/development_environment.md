# Development Environment

- Git configuration
- Local repository location
- VS Code settings
- MySQL connection
- What work can be done here

## macOS Home Computer
- Git configuration
- Local repository location
- MySQL installation
- MySQL Workbench configuration
- Python environment
- SSH keys (if used)

## Git Workflow

1. Commit changes at work
2. Push to GitHub
3. At home:
   git pull
4. Continue working
5. Commit
6. Push
7. Back at work:
   git pull

## First-Time Setup Checklist

- [ ] Clone repository
- [ ] Configure Git username/email
- [ ] Verify remote origin
- [ ] Test push/pull
- [ ] Verify MySQL connection
- [ ] Verify project opens correctly
